GIGGLE CODE

Fact about Giggle

A passionate code page from Vietnamese girl who is studying abroad in USA. Giggle Code offers lesson to train kids to learn coding with or without parents' support.

At Giggle Code, we impart the next-generation coding skill through the cup model to understand how robot works. After that, using Sphero 2.0, which is one of the best robot toys, for kids to learn coding.

Giggle Code equips students with skills in innovation and coding fundamentals. We explore innovative ways to educate students holistically and comprehensively - imparting our values in out-thinking, out-designing, and out-creating for the future.
